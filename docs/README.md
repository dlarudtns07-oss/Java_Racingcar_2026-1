# 자동차 경주 게임 기능 목록

## 기능요구사항
- [ ] 자동차 이름을 쉼표(,) 기준으로 구분하여 입력받기
- [ ] 각 자동차에 이름을 부여하기 (5자 이하 검증)
- [ ] 시도할 횟수 입력받기
- [ ] 0~9 사이의 무작위 값을 구하여 4 이상일 경우 전진하기
- [ ] 차수별 실행 결과 출력하기 (이름 : ---)
- [ ] 우승자 판별 및 출력하기 (공동 우승 시 쉼표로 구분)

## ⚠️ 예외 처리 
- [ ] 자동차 이름이 5자를 초과하는 경우
- [ ] 자동차 이름이 빈 값인 경우
- [ ] 시도 횟수가 숫자가 아닌 경우
- [ ] 시도 횟수가 음수인 경우
